# Online Store

Проект: создание онлайн-магазина.

## Стек используемых знаний
* Java 17
* Инкапсуляция
* ООП: наследование, абстрактные классы
* ООП: полиморфизм, интерфейсы
* Исключения в Java
* Java Collections Framework: List, Map, Set

*Примечание: Проект дорабатывается согласно требованиям и графика от заказчика*
